The requested domain is used by any active container :(

How do I fix this?

  1. First check your typing and check that the project you wanted exists.
  2. Then check that the requested project is started by running: 
$ kdev up

Is this a new domain name?

If is new domain for existing project you need edit open docker-compose.yml from your project folder and add following: 

    web:
        environment:
            VIRTUAL_HOST: example.local
            HTTPS_HOST: example.local

If this doesn't help look into the troubleshooting guide ( in Github Pages )
